# @theconvospace/sdk

SDK containing all the Hooks and Components required to build on [theconvo.space](https://theconvo.space)

## Installation

```bash
yarn add @theconvospace/sdk
npm i --save @theconvospace/sdk
pnpm i --save @theconvospace/sdk
```
